## About The Intake

Button Binding (demo/testing): 
* y - pivot ascending.
* a - pivot descending. 
* x - roller inwards. 
* b - roller outwards. 
* right_bumper - full sequence. 

Button Binding (official): 
* x - intake down while hold, back up and ready for the arm to grab the game piece from the intake.
